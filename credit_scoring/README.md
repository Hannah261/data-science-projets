Ceci est une analyse de rique sur le credit scoring.
Le modèle n'est pas encore à point (présente un overfitting avec peut-être risque de leakage) avec une seule première évaluation. 
Cependant le but n'est pas de chercher la meilleur performance brute mais de montrer le processus d'analyse du problème et bien maintrisé chaque étape.
