Ceci est une analyse de risque sur le credit scoring.
Le modèle n'est pas encore à point (présente un overfitting avec peut-être risque de leakage) avec une seule première évaluation. 
Cependant le but n'est pas de chercher la meilleure performance brute mais de montrer le processus de l'analyse du problème et bien maîtrisé chaque étape. 


## Install all dependances

```
pip install -r requirements.txt
```
