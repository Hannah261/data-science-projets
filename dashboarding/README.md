## Install all dependances (les dépendances)

```
pip install -r requirements.txt
```

## Run the app (exécution)

```
streamlit run dashboard_app.py
```

## Fonctionnalité

```
Dans cette version (sans base de données), la fonctionnalité de sauvegarde du dashboard personnalisé n'est pas inclus. 
Cependant, le test de création de dasboard est tout à fait possible.
```
