# Projet 1 : Credit scoring
Montre le processus d'analyse du problème de credit scoring de l'analyse statistique à la transformation métier avec modélisation prédictive (XGBoost).

# Projet 2 : Dashbording
Prototype de dashboard de pilotage centralisé qui ets personalisable avec des prédictions basiques intégrées.

# Projet 3 : Détection de fraudes financières (modèle unifié multi-fraude)
Système de détection de plusieurs types de fraudes qui utilise une approche hybride (Autoencoder, Kmeans, LightGBM) pour détecter et prédire le type précis de fraude.
